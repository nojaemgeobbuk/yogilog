앱 소개

요기로그는 요가 수련의 과정을 체계적으로 기록하고 아사나별 성장 단계를 추적하는 전문 요가 트래커입니다. 수련의 리듬을 시각화하고 나만의 시퀀스를 관리하며 더 깊은 수련의 세계로 나아가보세요.

자주 묻는 질문 (FAQ)

Q. 내 데이터는 어디에 저장되나요?

요기로그는 사용자의 프라이버시를 위해 모든 데이터를 서버가 아닌 **사용자의 기기(로컬)**에만 저장합니다. 외부로 데이터가 유출될 걱정 없이 안심하고 기록하세요.

Q. 앱을 삭제하면 기록도 사라지나요?

네, 모든 데이터가 기기 내에만 저장되므로 앱을 삭제하면 기록도 함께 삭제됩니다. 기기 변경이나 데이터 보존을 위해 설정 메뉴의 **'데이터 내보내기'** 기능을 통해 주기적으로 백업 파일을 보관해 주시기 바랍니다.
